Flight Logging Tips
Time Calculations
  • Total time is auto-calculated from departure/arrival times
  • Night time: enter in HH:MM format (e.g., 01:30 for 1.5 hours)
  • Instrument time: enter in HH:MM format (e.g., 00:45 for 0.75 hours)
  • Cross-country time: enter in HH:MM format (e.g., 02:00 for 2.0 hours)
  • All time components are validated against total flight time
Pilot Roles
  • PIC: Pilot in Command
  • SIC: Second in Command
  • Dual: Instruction received
  • Solo: Flying alone
